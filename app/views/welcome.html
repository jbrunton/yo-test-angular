Welcome

This is a demo single page app.

  • It's built using Yeoman, Grunt, Bower and Angular.
  • It's styled using Foundation (so it's responsive) and Sass.
  • It's unit testable, with continuous integration running on Travis

For more details see the repository on Github. Try it out below.

Here's the first sample from the Angular website:

Hello {{yourName}}!

Here's a simple filter which returns the fibonacci number for its input:
fib 0 {{ 0 | fib}}
fib 1 {{ 1 | fib}}
fib 5 {{ 5 | fib}}
fib 9 {{ 9 | fib}}

To run it locally, you'll need Node.js installed. (If you don't have it, check out the instructions on the wiki.)

Then run:

git clone git@github.com:jbrunton/yo-test-angular.git && cd yo-test-angular
npm install
bower install
